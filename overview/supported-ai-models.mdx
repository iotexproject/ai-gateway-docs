## Query Models Programmatically

```bash
curl https://gateway.iotex.ai/v1/models \
  -H "Authorization: Bearer your-api-key"
```

The response includes model metadata:

```json
{
  "object": "list",
  "data": [
    {
      "id": "gemini-2.5-flash",
      "object": "model",
      "created": 1677858242,
      "owned_by": "google"
    }
  ]
}
```
